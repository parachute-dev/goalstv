
# Goals TV

Goals TV Application to render Ads / League Information / Tournament Information / Kids Parties 


## Run Locally

Clone the project

```bash
  git clone https://github.com/parachute-dev/goalstv goals-tv
  cd goals-tv
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run start
```

Package / build the app

```bash
  npm run package
```
